# ==============================================================
# PIN NOTES — why each version is pinned where it is
# ==============================================================
#
# google-generativeai 0.8.3
#   Last stable pre-1.x SDK.  Works with gemini-1.5-flash and
#   gemini-1.5-pro.  The 1.x series changed the entire API surface.
#
# opencv-python-headless 4.10.0.84
#   Headless variant has no GUI dependencies — safe on servers,
#   Railway, Render, Docker, Heroku.  On a local desktop you can
#   swap to opencv-python at the same version number.
#
# Pillow 10.4.0
#   Pillow 11+ renamed LANCZOS → Resampling.LANCZOS and removed
#   several constants used by pdf2image and older cv2 glue code.
#   Pinned to 10.x for safety.
#
# numpy 1.26.4
#   OpenCV 4.10 requires numpy < 2.  numpy 2.x changed the C ABI
#   and breaks cv2 at import time.
#
# pypdf 4.3.1
#   pypdf 5.x changed PdfReader, PageObject, and stream internals.
#   Pinned to 4.x until policy_kb.py is updated for 5.x.
#
# reportlab 4.2.5
#   The PDF report uses platypus/flowables API stable since 3.x.
#   4.2.5 is the latest tested release.
